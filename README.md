## Problem Statement
Many government welfare schemes fail to reach eligible citizens due to lack of awareness, complex eligibility rules, and confusing application processes.

## Solution
Scheme Saathi is an AI-powered chatbot that helps users discover welfare schemes they are eligible for by understanding simple, natural language inputs and matching them with scheme criteria.

## How It Works
1. User enters their situation in simple language
2. NLP extracts key details (age, occupation, gender, etc.)
3. Eligibility engine matches relevant schemes
4. User receives benefits and official application links

## Tech Stack
- Frontend: Streamlit
- Backend Logic: Python
- NLP: Rule-based + keyword extraction
- Data: JSON-based scheme database
